I3

Горячие клавиши:

alt+Enter открывает терминал

alt+d запускает dmenu (меню сверху экрана, которое по мере ввода с клавиатуры названия приложения предлагает варианты для запуска)

alt+Shift+Q закрывает активное окно

alt+v включает режим вертикального тайлинга (экран будет делиться горизонтально)

alt+h включает режим горизонтального тайлинга (экран будет делиться вертикально)

alt+w включает режим вкладок (каждое окно на рабочем столе занимает весь экран, сверху видны вкладки)

alt+s включает стековый режим (заголовки окон один под другим, каждое окно занимает весь экран)

alt+e возвращает стандартный режим

alt+Shift+Space переключает окно в режим плавающего и обратно

alt+Left/Right/Up/Down перемещает фокус в пределах рабочего стола

alt+Shift+Left/Right/Up/Down перемещает текущее окно в пределах рабочего стола

alt+1 и т.д. переключает на рабочий стол с указанным номером

alt+Shift+C читает настройки из файла конфигурации

alt+Shift+E выходит из i3wm на экран ввода имени пользователя и пароля
