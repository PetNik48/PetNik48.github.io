Openbox

Горячие клавиши:

Win + c закрыть текущее окно (замена Alt + F4);

Win + m изменить размер текущего окна, чтобы оно занимало все доступное пространство рабочего стола;

Win + l свернуть (минимизировать) текущее окно;

Win + Space развернуть (максимизировать) текущее окно на весь экран или восстановить его размер, если оно уже развернуто;

Win + Enter поместить текущее окно в центр рабочего стола;

Win + ; переместить текущее окно на следующий монитор (конечно, если мониторов несколько);

Ctrl + Alt + Shift + h изменить размер текущего окна в левую сторону;

Ctrl + Alt + Shift + l изменить размер текущего окна в правую сторону;

Ctrl + Alt + Shift + k изменить размер текущего окна в нижнем направлении;

Ctrl + Alt + Shift + j изменить размер текущего окна в верхнем направлении;

Ctrl + Win + h сдвинуть текущее окно влево;

Ctrl + Win + l сдвинуть текущее окно вправо;

Ctrl + Win + k сдвинуть текущее окно вниз;

Ctrl + Win + j сдвинуть текущее окно вверх.
