MacOS

MacOS – это проприетарная операционная система с закрытым исходным кодом, основанная на Unix. Разрабатывается компанией Apple как сопутствующее программное обеспечение для собственной продукции. Устройства других производителей, согласно пользовательскому соглашению, использовать данную систему не имеют права.

История развития:

Первый компьютер Macintosh на базе Mac OS появился в далёком 1984 году. Самая первая версия операционной системе Apple, System 1.0 начала классическую серию Mac OS. System 7.0, 1991 год - Система получила первое глобальное обновление за 7 лет существования - цветной интерфейс. Mac OS 9 - последнее крупное обновление классической линейки Mac OS. Система поначалу имела индекс 8.7. Mac OS X 10.5 Leopard 2007 год - В Apple вели разработку и поддержку Mac OS X Leopard параллельно с разработкой iPhone OS для первого iPhone. Несмотря на это обновление Mac OS получилось крупным. macOS 10.12 Sierra 2016 год - Появился голосовой ассистент Siri из iOS, а браузер Safari получил поддержку платежного сервиса Apple Pay. macOS 10.14 Mojave 2018 год - Появилось множество стандартных приложений из мобильной версии iOS: Акции, Новости, Дом, диктофон. Появилась долгожданная темная тема. Обновлён магазин приложений Mac App Store.
